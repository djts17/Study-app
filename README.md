# 📚 Smart Study App – Class 10 NCERT
## Setup Guide for Non-Techies (Step by Step)

---

## WHAT YOU HAVE
All these files go into GitHub Pages for FREE hosting:
- `index.html` — The main app
- `books_data.js` — All chapter content (NCERT Class 10)
- `manifest.json` — Makes it installable on Android
- `sw.js` — Offline support
- `icon-192.png` / `icon-512.png` — App icons

---

## STEP 1: Create GitHub Account (5 mins)
1. Go to: https://github.com/signup
2. Enter your email, create password, choose username
3. Verify your email
4. Done!

---

## STEP 2: Create a New Repository (3 mins)
1. After login, click the GREEN "New" button (top left)
2. Repository name: `studyapp` (type exactly this)
3. Make sure "Public" is selected
4. Check the box "Add a README file"
5. Click green "Create repository" button

---

## STEP 3: Upload All 6 Files (5 mins)
1. In your new repository, click "Add file" → "Upload files"
2. Drag & drop ALL these files at once:
   - index.html
   - books_data.js
   - manifest.json
   - sw.js
   - icon-192.png
   - icon-512.png
3. Scroll down, click green "Commit changes" button
4. Wait 30 seconds

---

## STEP 4: Enable GitHub Pages (2 mins)
1. Click "Settings" tab (top of your repository)
2. Scroll down to "Pages" (left sidebar)
3. Under "Source", select "Deploy from a branch"
4. Branch: select "main" → folder: "/ (root)"
5. Click "Save"
6. Wait 2-3 minutes

---

## STEP 5: Get Your App Link
1. Go back to "Settings" → "Pages"
2. You'll see: "Your site is published at https://YOUR-USERNAME.github.io/studyapp/"
3. COPY that link — this is your app URL!
4. Share this link with the student

---

## STEP 6: Get Your FREE Gemini API Key (2 mins)
1. Open: https://aistudio.google.com/app/apikey
2. Sign in with Google account
3. Click "Create API Key" → "Create API key in new project"
4. Copy the key (starts with AIza...)
5. Give this key to the student

---

## STEP 7: Student Opens the App (Android)
1. Student opens Chrome on Android
2. Goes to: https://YOUR-USERNAME.github.io/studyapp/
3. First time: paste the Gemini API key → tap "Start Learning!"
4. Chrome shows: "Add StudyApp to Home Screen" → tap ADD
5. App icon appears on home screen — opens like a real app!
